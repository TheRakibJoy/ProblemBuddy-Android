## Install

1. Download the APK on your Android device.
2. Allow "Install unknown apps" for your browser/file-manager.
3. Open the APK → Install.

APKs are signed with a debug key, so Android will warn you about the source. That's expected for side-loading.

Rebuilt automatically by `.github/workflows/publish-apk.yml` on every push to `main`.
